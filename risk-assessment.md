# 📊 Risk Assessment Report

## 1. Introduction

This report evaluates cybersecurity risks facing BrightMart, a small retail business with limited security controls.

---

## 2. Assets Identified

* Customer database
* Payment processing system
* Company website
* Employee laptops

---

## 3. Risk Analysis

| Risk              | Likelihood | Impact | Risk Level |
| ----------------- | ---------- | ------ | ---------- |
| Data breach       | High       | High   | Critical   |
| Malware infection | Medium     | High   | High       |
| Website downtime  | Medium     | Medium | Medium     |

---

## 4. Recommendations

* Enforce strong password policies
* Implement multi-factor authentication (MFA)
* Install antivirus software on all devices
* Deploy firewalls to protect network traffic
* Provide employee cybersecurity training

---

## 5. Conclusion

Addressing these risks will significantly improve the company’s security posture and reduce exposure to cyber threats.
